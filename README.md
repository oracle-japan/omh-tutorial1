# Oracle Mobile Hub ハンズオン - バックエンド開発編

## はじめに

Oracle Mobile OMH （以下 OMH）は、オラクルが提供する「Mobile Backend as a Service（MBaaS）」です。
エンタープライズ・モバイルのシンプル化を目指しています。
企業は OMH を使用することで、拡張性と可用性に優れたセキュアなモバイル・アプリを素早く作成および展開できます。
任意のバックエンド・システムに接続するモバイル・アプリケーションを構築してデプロイできます。
また OMH は、既存の IT システムの活用を簡素化にも役立ちます。

このハンズオンでは、OMH の基本機能を実際に体験できます。

## このチュートリアルの表記方法

このチュートリアルで使用している表記方法は次のとおりです。

|表記方法 |説明 |
|-------|----|
|**太字**|ボタン、各種フィールドのラベルなどの GUI 要素|
|*イタリック*|使用する環境などによって置き換える部分を表すプレースホルダー|
|`固定幅フォント`|実行するコマンド、URL、サンプルコード、入力するテキスト|

この資料は主にWindowsの構文を用いていますが、MacまたはLinux環境でのファイル名およびディレクトリ名などは特に明記がない場合は同一です。
